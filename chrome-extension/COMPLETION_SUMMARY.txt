INSTALLATION & TESTING
================================================================================

To load the extension:

1. Open Chrome → chrome://extensions/
2. Enable "Developer mode" (top right)
3. Click "Load unpacked" button
4. Select: /sessions/inspiring-friendly-einstein/channel-companion/chrome-extension/
5. Extension will appear in toolbar
6. Configuration:
   - Click extension icon → Settings (gear)
   - Enter API token
   - Toggle sites to activate
   - Click "Save Settings"
